本機啟動
1. 直接用 Python 內建伺服器（推薦，跨平台）:
   - 在專案根目錄執行：
     - Python 3: `python -m http.server 8000`
   - 在瀏覽器開啟：http://localhost:8000/last-mile-delivery.html

2. 或用 Node 工具（如 live-server）:
   - 安裝：`npm install -g live-server`
   - 啟動：`live-server --port=8000`
   - 開啟：http://localhost:8000/last-mile-delivery.html
